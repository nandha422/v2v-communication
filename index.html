Spectrum Frequency Range

  • Operating band: 880 MHz – 905 MHz
  • Six discrete channels at 5 MHz spacing
  • Channels: 880, 885, 890, 895, 900, 905 MHz

Power & SNR Readings

  • Power levels: –72 to –102 dBm (measured)
  • SNR range: 11 dB to 22 dB per channel
  • Higher SNR → Better channel quality
  • Free channels identified via energy detection
🔍

Channel Status Classification

  • Status: OCCUPIED / FREE per channel
  • Channels 880, 890, 900 MHz → OCCUPIED
  • Channels 885, 895, 905 MHz → FREE
  • Dynamic sensing updates every cycle
🧠

AI-Based Decision Logic

  • Best channel = FREE with max SNR
  • Backup channel = 2nd best FREE channel
  • Collision risk → switch to backup instantly
  • Interference → redirect through alternate path
🚗

V2V Communication Parameters

  • Vehicle speed input: 0–120 km/h (adjustable)
  • TTC threshold: <2s = High Risk, <4s = Medium
  • Closing speed determines collision imminence
  • Real-time risk scoring: 0, 0.7, or 1.0

Spectrum Sensing Method

  • Energy detection on each 5 MHz sub-band
  • Threshold-based occupancy classification
  • Fast switching (<50ms) for safety-critical data
  • Cognitive radio: sense → decide → transmit

📌 How It Works

⏱

TTC Calculation

Time-To-Collision = Distance ÷ Closing Speed. TTC < 2s triggers critical collision alert and emergency channel switch.

📻

Channel Switching

On collision detection, system instantly switches from primary to backup channel to broadcast emergency braking signals.

⚡

Interference Handling

When speeds are nearly equal (<0.5 m/s difference), interference is detected and backup channel is activated automatically.

✅

Normal Operation

Best FREE channel (highest SNR = 885 MHz) used for routine V2V telemetry. Continuous monitoring of all 6 channels.

📌 Emergency Logic Explained

🚨

Same Lane Detection

Both vehicles start in the same lane. Ambulance approaches from behind — V2V detects shared lane and computes TTC.

➡

Lane Override Command

TTC < 5s triggers "Move RIGHT" command broadcast. At TTC < 2s, "MOVE NOW — EMERGENCY" is issued via backup channel.

📻

Dedicated Emergency Channel

Emergency V2X uses backup channel (905 MHz) to ensure interference-free delivery of priority clearance data.

✅

Lane Clear Confirmation

Once vehicle moves to adjacent lane and ambulance passes safe distance, vehicle returns to original lane. V2V reverts to primary.
